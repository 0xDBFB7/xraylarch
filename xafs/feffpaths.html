XAFS: Reading and using Feff Paths¶

For modeling EXAFS data, Larch relies heavily on calculations of theoretical XAFS spectra using FEFF. Being able to run FEFF and use its results is thus of fundamental importance for using Larch for fitting EXAFS spectra. While a complete description of FEFF is beyond the scope of this documentation, here we describe how to read the results from FEFF into Larch. The main interface for this is the feff_path() function that reads FEFF feffNNNN.dat file into a larch FeffPath object.
