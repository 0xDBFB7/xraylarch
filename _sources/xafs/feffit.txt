==============================================
XAFS: Fitting XAFS to Feff Paths
==============================================

.. module:: _xafs
   :synopsis: XAFS Feff Path and fitting functions

Here, we describe how to set up a fitting model to fit a set of FEFF
calculations to XAFS data.  The documentation so far has touched on many
aspects of this process, and those sections will be referenced here.
